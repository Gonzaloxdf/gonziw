Instalación
###########

Instalación
===========

- **Instalar Meteor**::

   $ curl https://install.meteor.com/ | sh

- **Clonar proyecto**::

   $ git clone https://github.com/argob/mi-argentina.git
   $ cd miargentina/
   $ meteor npm install
   $ cp settings.json.edit settings.json
