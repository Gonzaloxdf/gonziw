Instalación
###########

Requisitos
==========

- Node
- MongoDB ()

Instalación
===========

- **Instalar Meteor**::

   $ curl https://install.meteor.com/ | sh

- **Clonar proyecto**::

   $ git clone git@git.argentina.gob.ar:open-source/mi-argentina.git
   $ cd miargentina/
   $ meteor npm install
   $ cp settings.json.edit settings.json
